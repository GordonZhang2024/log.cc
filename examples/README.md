# log.cc example

This is the example for log.cc

## Build
### Requirements
- C++ compiler (g++, clang++, ...)
- CMake

**Note that MSVC Compiler has not been tested, but it should work as well.**
### Build
```bash
$ cmake -B build
$ cd build/
$ make
```
Then, you will see the executable file called `example`

Type command `./example` to run it.
